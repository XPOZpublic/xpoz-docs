Want to see what Xpoz returns before you sign up? Mint a **trial token** and start pulling social data in a single command. No account, no credit card, no waiting on approval.

## When to use this

A trial token is the fastest way to evaluate Xpoz — perfect for kicking the tires in a notebook, wiring up a quick proof-of-concept, or letting an agent try a real query during setup. For anything beyond a quick look (full result sets, exports, live data), grab a free access key instead.

## Get started

<Tabs>
  <Tab title="cURL">
    Mint a trial token, then use it as your API key:

    ```bash
    # 1. Mint a trial token (valid 5 days)
    TOKEN=$(curl -s -X POST https://api.xpoz.ai/api/trial/token | jq -r .data.accessKey)

    # 2. Use it like any access key
    curl -X POST https://mcp.xpoz.ai/mcp \
      -H "Authorization: Bearer $TOKEN" \
      -H "Content-Type: application/json" \
      -d '{"jsonrpc":"2.0","method":"tools/call","params":{"name":"getTwitterUser","arguments":{"identifier":"elonmusk","identifierType":"username"}},"id":1}'
    ```
  </Tab>
  <Tab title="CLI">
    ```bash
    export XPOZ_API_KEY=$(curl -s -X POST https://api.xpoz.ai/api/trial/token | jq -r .data.accessKey)
    xpoz-cli twitter get_user --identifier elonmusk
    ```
  </Tab>
  <Tab title="MCP (Claude Code)">
    ```bash
    TOKEN=$(curl -s -X POST https://api.xpoz.ai/api/trial/token | jq -r .data.accessKey)
    claude mcp add xpoz-mcp https://mcp.xpoz.ai/mcp \
      -t http \
      -H "Authorization: Bearer $TOKEN"
    ```
  </Tab>
</Tabs>

The mint endpoint needs no auth and hands back a token that starts with `TRIAL`:

```json
{
  "success": true,
  "data": { "accessKey": "TRIAL...", "expiresInSeconds": 432000 },
  "message": "Trial access token created."
}
```

Drop the `TRIAL...` token in wherever an access key goes — the SDK `apiKey`, the `XPOZ_API_KEY` env var, or a `Bearer` header.

## Upgrading is a no-op

A trial token returns the same fields, in the same shape, as a full access key. When you move up, your code stays exactly as it is — replace the token and keep going.

## What's capped

Trial tokens are deliberately scoped down:

| | Trial token | Access key |
|---|---|---|
| Data tools (read) | ✅ | ✅ |
| Results per call | 5 (cached) | Full, paginated |
| Pagination | ❌ | ✅ |
| CSV export | ❌ | ✅ |
| Live crawling | ❌ | ✅ |
| Validity | 5 days | Ongoing |

## Ready for more

Hit a cap, or done evaluating? [Grab a free access key](https://xpoz.ai/get-token) — still no credit card. Auth stays a `Bearer` token, and the request and response shapes are unchanged.
