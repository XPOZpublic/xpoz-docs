Claude and ChatGPT can both read live social media data through an MCP (Model Context Protocol) server. With Xpoz, the connection is a hosted URL plus an access key: add `https://mcp.xpoz.ai/mcp` as a connector, authenticate, and your assistant can search live Twitter/X, Instagram, Reddit, and TikTok data in natural language. Setup takes about 2 minutes and requires no code, no scraping, and no platform API keys.

"Live" here means the data behind the connector is continuously collected: billions of posts and users across the four platforms, refreshed as new content appears, with a rolling historical window. Your assistant queries that index directly instead of relying on its training data.

## Connect Claude to live social media data

Claude supports MCP connectors in three places. Each takes a minute or two:

<Tabs>
  <Tab title="Claude.ai (web)">
    1. Open **Settings** > **Connectors** (paid Claude plans)
    2. Click **Add custom connector**
    3. Enter the URL `https://mcp.xpoz.ai/mcp`
    4. Authenticate when prompted (sign in with Google, or use your access key)

    The Xpoz tools appear in Claude's tool menu in every new conversation.
  </Tab>

  <Tab title="Claude Desktop">
    1. Go to **Settings** > **Connectors**
    2. Click **Add custom connector**
    3. Enter the URL `https://mcp.xpoz.ai/mcp` and authenticate

    Desktop and web share the same connector flow; a connector added in one is available in the other.
  </Tab>

  <Tab title="Claude Code">
    Run one command in your terminal:

    ```bash
    claude mcp add xpoz-mcp https://mcp.xpoz.ai/mcp -t http -H "Authorization: Bearer YOUR_API_KEY"
    ```

    Claude Code discovers all 48 tools on the next conversation.
  </Tab>
</Tabs>

## Connect ChatGPT to live social media data

ChatGPT connects to MCP servers through developer mode, available on paid ChatGPT plans (see [OpenAI's developer mode guide](https://help.openai.com/en/articles/12584461-developer-mode-and-mcp-apps-in-chatgpt) for current plan availability):

1. Open **Settings** > **Connectors**, and under **Advanced** enable **Developer mode**
2. Back in **Connectors**, click **Create**
3. Name the connector (for example `Xpoz`), set the server URL to `https://mcp.xpoz.ai/mcp`, and complete authentication
4. In a new chat, open the **+** menu, choose **Developer mode**, and enable the Xpoz connector for that conversation

Then ask ChatGPT the same kinds of questions:

```
Summarize the top complaints about [brand] on Twitter and Reddit this week
```

```
Find Instagram posts mentioning [product] with the highest engagement this month
```

## Connect Gemini CLI to live social media data

Add the server to `~/.gemini/settings.json`:

```json
{
  "mcpServers": {
    "xpoz": {
      "uri": "https://mcp.xpoz.ai/mcp",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}
```

Restart Gemini CLI to pick up the configuration. Full per-client steps for Cursor, N8N, and other MCP clients are in [Installation](/mcp/installation).

## What you can do once connected

The connector exposes 48 tools across Twitter/X, Instagram, Reddit, and TikTok: user lookup and search, post search by keyword, comments, engagement, subreddit discovery, hashtag discovery, and continuous [tracking](/tracking) of keywords and accounts. Your assistant picks the right tool from your question; you never call tools by name.

## Frequently asked questions

### Can ChatGPT access live social media data?

Yes. ChatGPT cannot browse Twitter/X, Instagram, Reddit, or TikTok on its own, but with an MCP connector like Xpoz added through developer mode it queries a continuously collected index of those platforms and answers with current posts, comments, and profiles.

### Can Claude search Twitter or Reddit?

Yes. Claude's built-in web search does not reach most social platforms, but with the Xpoz MCP connector added in Settings > Connectors, Claude searches live Twitter/X, Reddit, Instagram, and TikTok data directly and can filter by keyword, author, date, and engagement.

### What is an MCP server?

MCP (Model Context Protocol) is an open standard that lets AI assistants call external tools. An MCP server hosts those tools; the assistant discovers and calls them automatically. Xpoz's MCP server is hosted (nothing to install) and exposes 48 social media data tools to any MCP-compatible assistant.

### Do I need API keys for each social platform?

No. One Xpoz access key covers all four platforms. Xpoz collects the data on its side; you never register developer accounts with Twitter/X, Meta, Reddit, or TikTok.
